Pocket Emacs

Running Emacs on your smartphone, PDA or other tiny computer.

My Pocket Emacs device is the Gemini PDA, an Android smartphone with an integrated hardware keyboard.

Emacs, India and Geminis -- Pocket Emacs goes to India

Slides for a talk given to the Berlin Emacs user group in December 2018.

The slides were prepared with help of the following software, all of which runs natively on Android in the Termux environment.

  • Termux, an Android app providing a Linux environment.
  • Emacs 26.1
  • Spacemacs, a nice set of curated packages for Emacs which can also be used in Emacs' Vim mode.
  • Org mode, the Emacs-hosted killer app.
  • reveal.js, an HTML presentation framework
  • Org-Reveal, a reveal.js exporter for Org mode

The presentation ran directly on the Gemini in a web browser. I also demonstrated using Emacs on the Gemini. The Android screen was streamed via Scrcpy over USB to my MacBook and thence to the data projector. There are other ways to achieve this, for example Vysor which works over Wifi or a direct connection to the projector from the Gemini's HDMI dongle.
